# Personal Portfolio Website

This simple, responsive portfolio website template helps you showcase your skills and projects when applying for junior web developer roles. It uses plain HTML, CSS, and vanilla JavaScript so it works without any framework.

## Structure

- **index.html** — Main page with sections for About, Projects, Skills, and Contact.
- **style.css** — Stylesheet for layout, colors, and responsive design.
- **script.js** — Small script to toggle the mobile navigation menu.

## Customization Steps

1. **Update Your Name**
   - In `index.html`, replace all occurrences of `Your Name` with your actual name.
2. **Write About Yourself**
   - Under the **About Me** section, add a concise biography, including your education and interests.
3. **Describe Your Projects**
   - Replace the placeholder project entries with a summary of your real projects (e.g., Pandan Kitchen, Aung Market). Include a short description and a link to each GitHub repository.
4. **List Your Skills**
   - Update the skills list in the **Skills** section to reflect your most relevant technical skills.
5. **Add Contact Links**
   - Replace the placeholder email and LinkedIn URLs in the **Contact** section with your actual contact details.
6. **Commit and Upload**
   - If you use GitHub, initialize a new repository and push these files. Include images or screenshots of your projects if you have them.
7. **Deploy**
   - You can deploy the site using a static site hosting service like GitHub Pages, Netlify, or Vercel. Copy your repository URL into the platform to get a live demo link.

This template is a starting point. Feel free to adjust the styling, add new sections, or integrate it with frameworks like React once your environment is ready.
